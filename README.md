# @enveloppe/api-test

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

Don't forget to create an `.env` file with the following content:

```env
GITHUB_TOKEN=
OWNER=
REPO=
BASE_BRANCH=
```

This project was created using `bun init` in bun v1.1.39. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
